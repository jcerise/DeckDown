# Welcome to Deckdown

A terminal-based presentation framework

---

## Features

- Markdown-powered slides
- Syntax-highlighted code blocks
- Interactive code editing
- Progress tracking
- Customizable themes

---

## Code Example

Here's a simple Python snippet:

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"

print(greet("World"))
```

---

## Executable Code

Press `Ctrl+R` to run this block:

```python exec
import math
print(f"Pi is approximately {math.pi:.6f}")
print(f"Euler's number is approximately {math.e:.6f}")
```

---

## Interactive Code

Edit and run this code live with `Ctrl+E`:

```python interactive
# Try changing the values!
x = 10
y = 20
print(f"{x} + {y} = {x + y}")
```
